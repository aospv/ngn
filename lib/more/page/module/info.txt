Немного о Мудулях Разделов

Модуль раздела представляет из себя несколько классов разных видов. Это:
- Ddo-класс для сайта
- Ddo-класс для админки
- Pmi (Page Module Installer). Класс, устанавливающий модуль
- Pma (Page Module Action). Различные действия специфичные для модуля. Вызываются в различных частях системы
  и др...
  
Модули поддерживают наследование всей группы классов с помощью наследования класса Pmi.
Таким образом Pmi-класс определяет предка модуля. Если из группы классов модуля, какой-то класс
будет отсутствовать, вместо него будет получен класс предка.
